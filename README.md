## API Reference

#### Получить список товаров

`
  GET /items
`

#### Получить один товар

`
  GET /items/${id}
`

#### Создать товар

`
  POST /items
`
#### Обновить товар

`
  PUT /items/${id}
`

#### Удалить товар

`
  DELETE /items/${id}
`
